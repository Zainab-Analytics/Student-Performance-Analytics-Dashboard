# Student Performance Analytics Dashboard

## Project Overview

This project analyzes student performance, attendance, and engagement data to identify academic trends, monitor learning outcomes, and support data-driven decision making. The dashboard provides insights into student performance, batch effectiveness, attendance patterns, and at-risk learners through interactive visualizations and KPI reporting.

## Business Problem

Educational institutions often struggle to identify students who require academic intervention and to measure the effectiveness of learning programs. This project aims to leverage student data to uncover performance trends, evaluate attendance impact, identify at-risk students, and support strategic academic planning.

## Objectives

* Monitor overall student performance and engagement.
* Identify high-performing and low-performing batches.
* Analyze the relationship between attendance and academic outcomes.
* Detect at-risk students for early intervention.
* Support academic decision-making through KPI tracking and dashboard reporting.

The dashboard enables educational organizations to track student performance, improve retention strategies, identify learning gaps, and make informed decisions using actionable insights derived from academic data.
